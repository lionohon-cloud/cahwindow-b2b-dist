# 청암홈윈도우 B2B 견적 프로그램 — 완전 명세서

> 이 문서를 Claude에 입력하면 동일한 B2B 견적 프로그램을 생성할 수 있습니다.
> Google Apps Script (Code.gs) + HTML (Index.html) 기반, PC 전용 웹앱입니다.

---

## 1. 시스템 아키텍처

### 1.1 기술 스택
- **프론트엔드**: 단일 HTML 파일 (Index.html), 순수 JavaScript (프레임워크 없음)
- **백엔드**: Google Apps Script (Code.gs)
- **데이터**: Google Spreadsheet (자동 생성)
- **배포**: GAS 웹앱 (`doGet` → `HtmlService.createHtmlOutputFromFile('Index')`)
- **통신**: `google.script.run` (GAS 클라이언트-서버)

### 1.2 외부 API 연동
- **원가 API** (`COST_API_URL`): 별도 GAS 웹앱에서 원가 시트 데이터를 JSON으로 제공
- **로그인 API** (`AUTH_API_URL`): 사용자관리 시트의 별도 GAS 웹앱, GET 방식
- **SSO**: 기존 사용자관리 시트 ID로 직접 접근

### 1.3 자동 생성 구조 (Google Drive)
```
📂 청암B2B견적/
├── 📊 청암B2B견적_v1 (quotes, config, 자체영업고객, B2B거래처, B2B현장, 견적요청)
├── 📊 청암B2B계약확정_v1 (계약확정_요약, 계약확정_품목, 계약확정_추가항목, 계약확정_정산)
├── 📊 청암B2B시공내역_v1
├── 📊 청암B2B할인권_v1 (할인권설정, 발행코드)
├── 📂 청암B2B견적이미지_v1
└── 📂 청암B2B작성페이지_v1
```

## 4. 가격 산출 공식 (핵심)

### 4.1 자평 계산
```
자평(JP) = ceil(W × H / 90000)
```
W, H: mm 단위

### 4.2 원가 조회 (`FP` 함수)
```
원가 = PD_COST[브랜드키][등급][자평 이상 행][품명 컬럼]
```
- `pdMode = 'perGrade'`: 등급별 원가 시트에서 직접 참조 (등급할인 0%)
- 자평이 시트 최대값 초과 시 -1 반환 → priceOver 처리

### 4.3 제품 단가 계산 (`CI` 함수)
```
1. 원가 조회: FP(브랜드키, 품명, 자평) → 원가
2. 브랜드 마크업: 원가 × (100 + mk.브랜드) / 100
3. 등급 할인: × (100 - gDisc) / 100  (perGrade면 0%)
4. B2B 할증: × b2bMarkup / 100  (기본 145%)
5. 브랜드×등급 마크업: × bgMk[브랜드][등급] / 100  (기본 PRESTIGE:120%, 나머지:100%)
6. GR 별도 할증: grLine.surcharge가 있으면 추가 곱셈
```

**최종 단가 공식:**
```
단가 = round(원가 × (100+mk) / 100 × (100-gDisc) / 100 × b2bMarkup / 100 × bgMk / 100)
```

### 4.4 시공비 (시공포함 모드만)
```
시공비 = round(자평 × 자평당시공비 × laborMk × 1)
```
- 이중창: `lp["이중창"] = 4,600원/자평`
- 단창: `lp["단창"] = 3,600원/자평`
- `laborMk = 1.15` (15% 할증)

### 4.5 철거비 (`calcDmFn`)
```
철거비 = round(Σ(선택된 철거 유형별 단가) × dmMk)
```
| 유형 | 단가 |
|------|------|
| AL | 10,000 |
| PVC | 20,000 |
| 세라믹 | 100,000 |
| 시스템창 | 100,000 |
| 돌와꾸 | 50,000 |
| 목틀 | 20,000 |

`dmMk = 1.15`

### 4.6 마감비
```
몰딩 = 50,000원 (건당)
사춤 = round(W(m) × 15,000)  (M당)
타일 = round(W(m) × 25,000)  (M당)
브라켓 = ceil(W / (50cm × 10)) × 3,000원
마감비 합계 = round(Σ × fnMk)
```
`fnMk = 1.15`

### 4.7 추가옵션 (aH, aL, aK — 할인 비적용)
**방충망** (등급별 자동 결정):
- PRESTIGE/GR → 안전방충망 (130,000원/개)
- SIGNATURE → 블랙스텐망 (시트 자평 자동조회)
- ECO → 스텐망 (시트 자평 자동조회)
- 방범방충망: 180,000원, 방범방충망(UP): +50,000원

**핸들** (등급별):
- PRESTIGE/GR → 무료 (0원)
- SIGNATURE/ECO → 이중창: 100,000원, 단창/공틀: 50,000원

### 4.8 항목별 총액
```
항목 총액 = (제품단가 + 추가옵션) × 수량 + (시공비 + 철거비 + 마감비) × 수량
```
- 시공비는 시공포함 모드에서만 계산
- 시공비는 브랜드 무관 (동일)

## 5. 할인 체계

### 5.1 할인 적용 원칙
- **제품가에만 할인** 적용, 추가옵션(방충망/핸들)은 할인 제외
- 시공비, 관리비, 물류비, 추가항목에는 할인 미적용
- 할인제외 품명 (라운드, PJ창 등): Admin에서 설정

### 5.2 할인권
- 정액: 제품가 비율로 배분 (`amt × ratio`)
- 율(%): 직접 적용 (`base × rate / 100`)
- 순차 복리 적용 (1번 → 2번 → 3번)
- `_discProdTotal`: 할인 대상 제품가 합계 (매 R() 사이클마다 초기화)
- `maxCpnRate = 5`, `maxCpnRateAlways = 20`, `minCpnQuoteAmt = 5,000,000`

### 5.3 B2B 카드 수수료 (할인 아닌 가산)
```
카드수수료 = round(견적금액(총) × cardSurcharge%)
```
- `cardSurcharge = 3` (기본 3%)
- 총금액 기준 (제품+시공+추가+관리비 전부)
- 개별 항목 단가에는 미반영, 최종 합계에서만 가산

### 5.4 견적 합계 산출
```
견적금액 = Σ(항목 총액) + 추가항목 합계 + 관리비 합계
공급가 = 견적금액 - 할인권 할인 + 카드수수료
VAT = round(공급가 × 0.1)
최종가 = 공급가 + VAT
```

## 7. 관리비 + 물류비 체계

### 7.1 관리비
```
기본관리비 = round(mgmtFee × exMk) = round(300,000 × 1.15) = 345,000
최소시공비 = 자평 ≤ minLaborJp(120)이면 minLaborFee(900,000) 추가
관리비합계 = 기본관리비 + 비공개정산 + 물류비 + (최소시공비)
```

### 7.2 물류비 A/B/C 권역
```
기준: 제품 원가 합계 (3사 중 최대값)
```

| 권역 | 기준금액 이상 | 비율 | 기준금액 미만 | 고정비 |
|------|-------------|------|-------------|--------|
| A | 3,500,000 | 8% | 미만 | 200,000 |
| B | 4,000,000 | 12% | 미만 | 500,000 |
| C | 4,000,000 | 17% | 미만 | 700,000 |

권역 판정: 주소의 시/도 기준
- B 권역: 경상북도, 경상남도, 강원특별자치도, 울산광역시, 부산광역시
- C 권역: Admin에서 추가 설정
- A 제외: 경상북도 구미시, 경북 구미시 등
- `logiMk = 1` (물류비 할증 배수)

### 7.3 미리보기/견적서에서 자평당 녹이기
```
녹일 금액 = 기본관리비(×할증) + 물류비
자평당 = 녹일 금액 ÷ 총 자평합계
각 제품 = 원래 제품가 + (자평당 × 해당 제품 자평)
```
- 작성 화면: 기존대로 관리비/물류비 별도 행
- 미리보기/견적서: 제품가에 녹여서 표시
- 나머지(비공개정산+최소시공비): "기타 관리비" 별도 행

## 21. 데이터 흐름 요약

```
[원가 API] ──fetch──→ PD_COST 주입
                         ↓
[품명 선택] → CI(item) → { lp, hp, kp, aL, aH, aK, lb, dm, fn } × qty
                         ↓
[견적합계] = Σ(제품+추가+시공) + 추가항목 + 관리비 + 물류비
                         ↓
[할인] = 할인권(제품가만) ← applyDiscToPrice()
[카드] = +수수료(총금액×3%)
                         ↓
[공급가] = 견적금액 - 할인 + 카드수수료
[최종가] = 공급가 × 1.1 (VAT)
```
